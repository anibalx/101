Formjacking 
é um ataque cibernético 
que permite a criminosos 
roubar informações bancárias de clientes 
diretamente de sites de compras online

Os atacantes inserem código JavaScript malicioso nas páginas de pagamento dos sites
Os criminosos conseguem interceptar os dados de cartões de crédito sem levantar suspeitas

é uma das principais causas de vazamentos de números de cartão de crédito, 
gerando graves prejuízos financeiros e multas milionárias

Para se proteger do formjacking, é possível
	Reforçar os processos de governança de segurança, incluindo todos os elementos de terceiros
	Garantir que a organização esteja sempre em dia com patches 
	Utilizar web application firewalls (WAFs) e subresource integrity (SRI) 
	Utilizar soluções de detecção e projeção automatizadas que scaneem web applications por anomalias
